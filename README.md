# TechFlow Task Manager

## Descrição

Projeto acadêmico desenvolvido para a disciplina de Engenharia de Software.

O sistema simula o desenvolvimento de uma solução de gerenciamento de tarefas para uma startup do setor de logística, utilizando metodologias ágeis, versionamento de código e práticas de controle de qualidade.

## Objetivo

Desenvolver um sistema capaz de permitir o gerenciamento de tarefas através das operações básicas de cadastro, consulta, edição e exclusão.

## Escopo Inicial

* Criar tarefas
* Listar tarefas
* Editar tarefas
* Excluir tarefas

## Metodologia Utilizada

Kanban

## Tecnologias Utilizadas

* Git
* GitHub
* Pytest
