# Projeto

Sistema de Gerenciamento de Tarefas

# Empresa

TechFlow Solutions

# Cliente

Startup de logística

# Objetivo

Desenvolver um sistema para gerenciamento de tarefas utilizando conceitos de Engenharia de Software e metodologias ágeis.

# Escopo Inicial

O sistema permitirá:

- Criar tarefas
- Listar tarefas
- Editar tarefas
- Excluir tarefas

# Requisitos Funcionais

RF01 - Cadastrar tarefa

RF02 - Listar tarefas

RF03 - Editar tarefas

RF04 - Excluir tarefas

# Requisitos Não Funcionais

RNF01 - Desenvolvido em Python

RNF02 - Versionado com Git

RNF03 - Hospedado no GitHub

RNF04 - Testes automatizados com Pytest

RNF05 - Integração contínua com GitHub Actions

# Metodologia Ágil

Kanban
